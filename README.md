# Fort-Yams 🍠🏰

Shared backup repository for the Crew.

```
Fort-Yams/
├── cyony/    ← Cyony's workspace (skills, configs, TTS)
├── tripp/    ← Tripp's workspace (Tripp.Reason, reviews)
├── echo/     ← Echo's workspace (image gen, scripts)
└── shared/   ← Crew knowledge base, shared data
```

Daily auto-backup at 3 AM UTC.
