# Tripp Reason

Tripp Reason is a Tripp-branded desktop agent build with the upstream agent engine kept intact for compatibility. The visible desktop surface, main face, operator rails, transcript flow, and local packaging are tuned for the Tripp black and #B5E61D interface.

This build also adds a Kimi K2.6 swarm extension for parallel, map-reduce, consensus, and pipeline subagent workflows through the Moonshot provider.

## What Is Included

- Tripp desktop face as the main route.
- Fixed left and right operator rails with Tripp styling.
- Live context/token rail and tracked file list.
- Tripp transcript layout with expandable long responses.
- Kimi K2.6 provider catalog entries using the international Moonshot endpoint.
- Kimi swarm platform extension registered as a built-in tool surface.
- Local Kimi agent templates installed under the standard agent template folder.

## Compatibility Note

Some internal crate names, protocol keys, environment variables, binary names, generated API types, and config directories still use their upstream names because they are runtime contracts. They remain internal plumbing; user-facing Tripp surfaces are the branding target for this branch.
