# Ledger Template

Use this format for ALL spending trackers. Copy and populate.

```markdown
# Eddie Mitchell — Property Spending Ledger
## [Project/Property Name]
### As of: [Date]

---

## 📍 PROPERTY NAME — Address

| Date       | Description                              | Debit      | Credit     | Balance     |
|------------|------------------------------------------|------------|------------|-------------|
|            | Work quoted                              | $0,000.00  |            | ($0,000.00) |
|            | Paid — cash/bank/Zelle                   |            | $0,000.00  | ($0,000.00) |
|            |                                          |            |            |             |
|            | **PROPERTY TOTAL**                       |            |            | **($0,000.00)** |

---

## 💰 BALANCE SUMMARY

| Category                                | Amount        |
|----------------------------------------|---------------|
| Total Spent (confirmed)                | $0,000.00     |
| Total Spent (estimated untracked)      | $0,000.00     |
| Still Owed (confirmed)                 | $0,000.00     |
| Still Owed (estimated)                 | $0,000.00     |
| **PROJECTED TOTAL**                    | **$0,000.00** |
```

## Dollar Format Rules
- ALWAYS `$0,000.00` format
- Comma every three digits: `$12,345.00` not `$12345`
- Two decimal places always: `$1,600.00` not `$1600`
- Never abbreviate: never `$4.5k` or `$12k`
